install debian
--------------
- choose usa **change later**
- select all defaults
- hostname: zz
- domain: home.arpa
- swap partition equal to ram
- do not set root password
- uncheck install of all additional environments and files

update debian
-------------
$ sudo apt update
$ sudo apt full-upgrade

install gnome
-------------
$ sudo apt install \
gnome-console \
gnome-disk-utility \
gnome-session \
gnome-tweaks

remove gnome apps
-----------------
$ sudo apt purge im-config malcontent yelp
$ sudo apt autoremove

install caskaydia mono nerd font
--------------------------------
$ wget -P ~/.local/share/fonts \
https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/CascadiaMono.zip \
&& cd ~/.local/share/fonts \
&& unzip CascadiaMono.zip \
&& rm -f CascadiaMono.zip LICENSE README.md \
&& fc-cache -f

--------

$ cd dotfiles
